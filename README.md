#  Banking System GUI (Java)

##  Project Description
This is a simple Banking System application developed using **Java Swing (GUI)**.  
It allows users to create accounts, login, and perform basic banking operations in a user-friendly interface.

-  Login System (Account Number + Name)
-  Create New Account
-  Deposit Money
-  Withdraw Money
-  Check Balance
-  Logout Functionality
-  Graphical User Interface (Java Swing)

---

##  Concepts Used
- Object-Oriented Programming (OOP)
- Java Swing (GUI Development)
- Event Handling
- Exception Handling
- Collections (HashMap)
